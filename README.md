# Heart to Home

A Vite React prototype for gifting health services in Kathmandu to loved ones in Nepal.

## Run locally

```bash
npm install
npm run dev
```

Then open the local URL printed by Vite.

## Build

```bash
npm run build
```

## Prototype flow

- Home dashboard explains the service and trust factors.
- Services tab lets users select HS1, HS2, and other dummy health services.
- Gift Now opens a recipient/sender form and stores the selected services in app state.
- Submitting the form opens a dummy payment summary with payment options.
- Contact Us shows placeholder owner contact information.
